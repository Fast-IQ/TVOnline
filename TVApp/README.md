# ТВ Приставка - Android TV приложение

Приложение для просмотра российских телеканалов на Android TV приставках.

## Функции

- 📺 Просмотр российских телеканалов (Первый канал, Россия 1, НТВ и др.)
- 🕐 Программа передач (EPG) с поддержкой разных часовых поясов
- 🎬 Информация о текущей передаче
- ⏪ Перемотка времени для архивных передач
- 🌐 Выбор языка интерфейса (Русский/English)
- 📊 4 режима качества видео (Минимум, Среднее, Максимум, Авто)
- 💾 Запоминание последнего канала
- 🎮 Полная поддержка управления с пульта

## Сборка проекта

### Требования

- Android Studio Arctic Fox или новее
- JDK 11 или новее
- Android SDK 34

### Шаги сборки

1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Подключите Android TV устройство или эмулятор
4. Нажмите Run (Shift+F10)

### Локальная сборка через командную строку

```bash
cd TVApp

# Сборка отладочной версии
./gradlew assembleDebug

# Сборка релизной версии
./gradlew assembleRelease

# Установка на подключенное устройство
./gradlew installDebug
```

APK файл будет создан в:
- `app/build/outputs/apk/debug/app-debug.apk`
- `app/build/outputs/apk/release/app-release-unsigned.apk`

## Управление с пульта

| Кнопка | Действие |
|--------|----------|
| D-PAD Вверх/Вниз/Влево/Вправо | Навигация по интерфейсу |
| OK/Enter | Выбор канала/опции |
| Back | Назад |
| Menu | Открыть настройки |
| Info (i) | Показать информацию о программе |
| Play/Pause | Пауза/Воспроизведение |

## Настройки

### Часовой пояс
- Регулировка от UTC-12 до UTC+12
- Влияет на отображение программы передач

### Качество видео
- **Минимум (240p)** - Для медленного интернета
- **Среднее (480p)** - Баланс качества и скорости
- **Максимум (1080p)** - Лучшее качество
- **Авто** - Автоматический выбор

### Язык интерфейса
- Русский
- English

## Технические детали

- **Язык**: Kotlin
- **Минимальная версия Android**: 5.0 (API 21)
- **Целевая версия Android**: 14 (API 34)
- **Видеоплеер**: ExoPlayer 2.19.1
- **Формат потоков**: HLS (.m3u8)

- Для работы требуется стабильное интернет-соединение
- Потоки предоставлены из открытых источников
